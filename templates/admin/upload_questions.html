📋 Excel File Format Requirements:

Your Excel file should have the following columns:

  • question_text - The question text
  • question_type - 'mcq' or 'theory' (optional, defaults to 'mcq')
  • option_a - Option A
  • option_b - Option B
  • option_c - Option C
  • option_d - Option D
  • correct_answer - The correct answer (A, B, C, D, or text for theory)

📝 Note: For theory questions, you can leave options empty and provide answer in correct_answer field.
